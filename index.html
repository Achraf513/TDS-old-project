About

The project consists of a very simple e-commerce site. A web application that provides the user with different product categories, which can be consulted, ordered and even give an opinion concerning a product already purchased.

E-commerce is a single page application website, fully coded from scratch, without the use of any open-source frameworks nor any toolkits technologies used: HTML5, JavaScript, TypeScript, Java and CSS

This project was simply made to understand and work around the MVP design pattern as part of our educational program in SUPCOM
